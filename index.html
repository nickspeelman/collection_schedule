Refuse Collection Is Confusing!

Tracking refuse collection in our neighborhood is difficult. Trash, recycling, and yard waste each follow different rules for holiday shifts, and they don’t always observe the same holidays.

Each cell represents a single day in 2025, with holidays and collection days color-coded. Below the date, a bar visually represents scheduled collections:

  • Light Blue → Holidays
  • Dark Blue → Trash
  • Dark Red → Recycling
  • Light Red → Yard Waste

If multiple collections fall on the same day, the bar is divided proportionally to show each collection type. If only one collection type occurs, the entire bar is filled with that color.
